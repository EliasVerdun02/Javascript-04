Sistema de promedios

Programa que permite el ingreso del apellido de 3 alumnos, y a partir de alli el ingreso de 3 notas correspondientes a cada alumno. Segun sus notas se calcula el promedio y se indica si aprobo o no la materia.
Para comenzar : empezar()
Para mostrar resultados : mostrar()

Funciones:

  • empezar(): Funcion para ejecutar el programa
  • mostrar(): Funcion para mostrar el promedio final de cada alumno ingresado y si aprobo la materia
  • mostrarAlumno(AlumnoBuscado): Funcion para mostrar la informacion de un alumno en espeficico a traves del parametro de la funcion mediante su nombre
  • mostrarAprobados(): Funcion para mostrar la informacion de los alumnos aprobados
  • mostrarDesaprobados(): Funcion para mostrar la informacion de los alumnos desaprobados

Ejemplo:

  • Alumno: Elias
  • Notas: 8,7,6
  • Promedio Final: 7
  • Estado: Aprobado
  • Alumno: Pedro
  • Notas: 8,7,7
  • Promedio Final: 7.33
  • Estado: Aprobado
  • Alumno: Juan
  • Notas: 5,4,3
  • Promedio Final: 4
  • Estado: Desaprobado
  • Alumno: Marcos
  • Notas: 8,8,4
  • Promedio Final: 6.67
  • Estado: Desaprobado
  • Alumno: Tito
  • Notas: 10,8,4
  • Promedio Final: 7.33
  • Estado: Aprobado
